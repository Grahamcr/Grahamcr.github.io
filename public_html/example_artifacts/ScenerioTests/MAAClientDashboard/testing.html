What is the Application Compatibility Rating System?

Ever been using an application over MyApps Anywhere an found that not all of the features work as expected? This can happen when not all of an application's features are optimized to work over MyApps (i.e. java applets).  In order to help our users get a sense if the application they are about to use has such problems we have come up with an Application Compatibility Rating. 


The Compatibility Rating is a combination of users rating the applications they use over MyApps and our own algorithm which helps determine compatibility.


Possible Compatibility Ratings:

Works Great on MyApps:

Some Users Have Reported Issues:

More Than 50% of Users Report Issues:

More Than 75% of Users Report Issues:

Application Has Known Issues:


How Can I Rate an Application I Use?

You can help us inform other users about an application's compatibility by rating applications you have used over MyApps via the MyApps Dashboard on your Computer.  Simply open the Dashboard, click on Give Feedback and rate applications you have used using the Five Star Rating system.

How Do I View an Application's Compatibilty Rating?

You can see the Compatibility Ratings of the applications you use by clicking Show Compatibility Rating in the What's Running on MyApps section of the MyApps Dashboard Home Tab.
